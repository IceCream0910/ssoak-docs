# 사용된 기술

쏙을 개발하기 위해 어떤 프로그래밍 언어들과 프레임워크 등 기술들이 사용되었을까요?

---

### HTML/CSS/JS
HTML과 CSS, JS는 웹 개발을 위해 반드시 필요한 기술입니다.

HTML은 '마크업 언어'로 화면에 구성요소들을 배치할 수 있는 수단이에요.
CSS(Cascading Style Sheets)는 언어가 아닌 스타일 문서입니다. 'HTML'에서 만든 요소들의 디자인을 정의해주는 문서라고 볼 수 있습니다.
JS는 Javascript(자바스크립트)를 말하는데, 이게 바로 프로그래밍 언어에 속한다고 볼 수 있습니다.
브라우저에서 다양한 일을 수행하고 서버에서 데이터를 가져와 HTML과 CSS로 구성한 화면에 값을 변경해주는 데 사용됩니다.

### jQuery
제이쿼리는 웹사이트에 자바스크립트를 쉽게 활용할 수 있도록 도와주는 오픈소스 기반의 자바스크립트 라이브러리입니다.
<Callout>
  라이브러리란? | 재사용이 필요한 기능으로 반복적인 코드 작성을 없애기 위해 언제든지 필요한 곳에서 호출하여 사용할 수 있도록 만들어진 것.
</Callout>

보다 쉽게 HTML에서 정의한 요소에 접근할 수 있도록 기능을 제공하고 있으며 ajax라고 하는 네트워크 관련 기능도 있어 채택하였습니다.
> 2020년 “웹사이트용 자바스크립트 라이브러리 사용 동향”에서 제이쿼리는 76%의 사용률로 압도적인 1위를 기록하고 있습니다.
그러나 최근 여러 최신 라이브러리들이 등장하면서 jquery의 위상은 낮아지고 있는 것도 사실입니다. 그러나 지속적인 업데이트가 이루어지고 있으며 여전히 많은 서비스에서 사용되고 있는 라이브러리입니다.

## 서버 & 데이터베이스

### Firebase 
구글에서 운영 중인 모바일 애플리케이션 개발 플랫폼으로 실시간 데이터베이스, 인증(로그인), 클라우드 메시징(푸시 알림) 등 다양한 기능을 제공합니다.
쏙의 커뮤니티 기능과 수행평가 리스트 기능에 데이터베이스와 로그인 기능 동작에 사용되고 있습니다.

## 안드로이드 앱

### Kotlin
안드로이드 앱을 개발하기 위한 프로그래밍 언어로는 Java와 Kotlin(코틀린)이 대표적입니다. 현재는 Java 대신 Kotlin이 공식 언어로 채택된 상태이며, Java와 유사하지만 보다 간결하고 효율적인 코드를 작성할 수 있습니다.
쏙의 안드로이드 앱 역시 Kotlin 기반으로 개발되고 있습니다.
